# Inspiration Folder

This folder exists to provide inspiration for future functionality in the main repository.

## Purpose

- Treat everything here as **reference material**, not a direct blueprint.
- Use these examples to spark ideas for what we might build next.

## Ground Rules

1. **Inspiration is not direction.**
   We should not change our core architecture, fundamental structure, or guiding design principles just to match what appears in this folder.

2. **Reference code may be imperfect.**
   The code here is not guaranteed to be optimal, production-ready, or aligned with our standards. Its value is in the ideas it suggests.

3. **Improve, don’t copy blindly.**
   If an idea is useful, adapt it thoughtfully to our repository’s patterns and quality bar. In many cases, there will be a better way to implement the same concept.
